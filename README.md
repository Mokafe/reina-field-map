# REINA'S FIELD MAP

![ChatGPT Image 2025年12月18日 18_48_30](https://github.com/user-attachments/assets/6d7f2171-2077-4790-b765-3b3d6848d79f)


強化学習（Q-learning / Double Q / Dyna-Q）が 4x4 のフィールドを学習していく様子を、3Dで可視化したデモです。

## 公開URL
https://mokafe.github.io/reina-field-map/

## 操作方法
- **START / PAUSE**：学習の開始・停止
- **RESET**：学習をリセット（Q値・統計・ログを初期化）
- **PATH**：現在のQ値から推定される「最適パス」を表示/非表示
- **POLICY**：各マスの「推奨行動（方策）」の矢印を表示/非表示
- **GRID**：床グリッドを表示/非表示

右下のスライダーで、探索率（ε）、学習率（α）、割引率（γ）、ノイズ(slip)などを調整できます。

## モードの違い（ひとこと）
- **Double Q**：Q値の過大評価を抑え、学習が安定しやすい手法です。
- **Dyna-Q**：経験から環境モデルを作り、疑似経験（planning）も使って学習を加速します。

## 小説とのリンク（世界観）

このビジュアライザは、4x4のFrozenLake（強化学習の迷路）を、小説の世界観に置き換えて表現しています。

- **Field（F）**：星見ケ里フィールド（観測が行われる場）
- **Start（S）**：レイナの観測開始地点（「意識マップ」の起点）
- **Garden Gate（G）**：父ヨウスケが示した「光の庭」への入口（到達目標）
- **Distorter（D）**：クロスフィールド社が設置した“意識を乱す”領域・装置（判断がぶれやすい）
- **Deep Water（W）**：川の深み／井戸のような危険領域（落ちるとセッション遮断＝大きなペナルティ）
- **ノイズ（slip）**：環境ノイズ／意識の乱れ（思った行動がズレる確率）

可視化の対応：
- **柱の高さ**＝そのマスの価値（max Q）
- **矢印**＝そのマスで推奨される行動（方策 / policy）
